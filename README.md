# Soalify: Aplikasi Pembuat Soal Otomatis

> A full-stack web application that automatically generates question-and-answer pairs from PDF documents using a Retrieval-Augmented Generation (RAG) architecture. Developed as part of a thesis research project.

## ✨ Features
- AI-powered automatic quiz generation from PDFs
- Interactive UI to review and edit generated questions
- Robust backend using FastAPI for rapid inference

## 🚀 Getting Started
1. Clone the repository.
2. Set up the frontend using `npm install` and `npm run dev` in the Next.js directory.
3. Set up the backend by installing Python dependencies and running `uvicorn`.

## 🛠️ Tech Stack
- **Frontend:** Next.js, TypeScript, Tailwind CSS
- **Backend:** Python, FastAPI
- **AI System:** RAG (Retrieval-Augmented Generation), Google Gemini, ChromaDB, LangChain
